# Proyecto AegisVault-Security

## Descripción General
Este proyecto consiste en el desarrollo de una aplicación web funcional que implementa técnicas criptográficas utilizando Python. La aplicación se desarrollará en un periodo de seis semanas por un equipo de tres estudiantes, siguiendo criterios de HCI (UX, UI, ID, UCD). El objetivo principal es reforzar el entendimiento teórico-práctico de la criptografía mediante programación y desarrollo de software seguro.

## Diseño de la Arquitectura
La arquitectura del sistema se divide en tres componentes principales:
1. **Front-end**: Interfaz de usuario que permite la interacción con la aplicación. Se desarrollará utilizando tecnologías web modernas para asegurar una experiencia de usuario óptima.
2. **Back-end**: Implementado en Python, utilizando un framework como Flask, Django o FastAPI para manejar las solicitudes de la API y la lógica del negocio.
3. **Base de Datos**: Se utilizará un sistema de gestión de bases de datos como MySQL, PostgreSQL o MongoDB para almacenar la información de los usuarios y datos sensibles.

## Diseño de la Base de Datos
La base de datos se diseñará para soportar un sistema CRUD completo, permitiendo la creación, lectura, actualización y eliminación de información sensible. Se incluirán tablas para usuarios, roles y registros de actividad, asegurando la integridad y seguridad de los datos.

## Elección de Bibliotecas Criptográficas
Se implementarán al menos tres técnicas criptográficas, que pueden incluir:
- **Cifrado Simétrico**: AES o DES para el cifrado de datos.
- **Cifrado Asimétrico**: RSA o ECC para la gestión de claves y cifrado de datos.
- **Hash Seguro**: SHA-256, SHA2, SHA3, bcrypt, y técnicas de salting para el almacenamiento seguro de contraseñas.
